# STM32-MINI-PROJECTS
Mini projects


1- STM32 Blue Pill – LED Blink

## Description
Blinking external LED using GPIO PA5 on STM32F103.

## Hardware
- STM32 Blue Pill
- ST-LINK V2
- LED + 220Ω resistor

## Software
- STM32CubeIDE
- Embedded C (HAL)

## What I learned
- GPIO configuration
- HAL functions
- Debugging with ST-LINK
